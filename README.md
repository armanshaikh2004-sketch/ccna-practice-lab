 Networking Lab - Cisco Packet Tracer

Overview
-This repository contains Cisco Packet Tracer labs for VLAN/VTP and Static Routing.

 VLAN & VTP Lab
- VLAN creation for departments
- VTP configuration (server/client)
- VLAN propagation between switches

 Static Routing Lab
- Router-to-router configuration
- Static route setup
- Network connectivity testing using ping

 Tools Used
- Cisco Packet Tracer

 Outcome
 - Practiced switching and routing concepts in a simulated enterprise network.
